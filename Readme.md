## Requirements:

### Node
  - Updated to version 0.12.4

### Babel
  - [Github-Babel](https://github.com/babel/babel)
  `npm install babel -g`

### Nodemon
  - [Github-Nodemon](https://github.com/remy/nodemon)
  `npm install nodemon -g`

## Running The API

  `npm install`
  `npm start`

## Testing the API

  `npm test`
